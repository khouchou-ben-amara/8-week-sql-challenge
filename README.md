# 8 Week SQL Challenge

This repository documents my work on the **8 Week SQL Challenge**, a structured
set of real-world SQL case studies designed to progressively build strong
data analysis skills.

The challenge covers a wide range of SQL concepts, from basic aggregations
to more advanced analytical logic, including joins, subqueries, and window
functions.
